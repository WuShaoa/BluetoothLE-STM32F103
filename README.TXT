实验现象:
低功耗从机（从设备），实现多种方式唤醒从设备。从设备模块在断连时，自动进
入睡眠，在睡眠时从设备处于广播待连接状态，可以被主设备或手机连接唤醒，
也可以通过UART串口和WKUP引脚唤醒。在连接时，按下KEYUP按键，发送数据给主设备。
在睡眠时，按下KEY1按键，通过WKUP引脚唤醒从设备。按下KEY0按键，通过UART串口发送数据唤醒从设备。
